# The Button Game
## A game to be played using an AVR. Who is the fastest to turn on a whole row of leds?!. The game is done using generic GPIO/LED/Button drivers.
# Hardware
        14 LED
        3 Push Buttons
        17 Pull 300 ohm Resistor
        25 Jumber
        AVR Atmega32
